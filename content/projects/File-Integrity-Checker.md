# File Integrity Checker – File Integrity Monitoring Tool

**File Integrity Checker** is a Python-based security tool designed to monitor and detect unauthorized changes to files. It works by generating cryptographic hashes for files and comparing them against previously stored baseline hashes to identify any modifications.

## 📅 Project Duration
**2023**

## ✨ Features

- Baseline hash generation for monitored files
- Detects file modification, deletion, and tampering
- Compares current file hashes with stored hash values
- Supports monitoring of critical files and directories
- Lightweight and easy-to-use CLI-based tool
