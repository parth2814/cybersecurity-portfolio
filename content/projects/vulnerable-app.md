# Vulnerable Web App

An intentionally vulnerable web application designed for security training and research purposes.

## Vulnerabilities

- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Insecure Direct Object References (IDOR)
- Broken Authentication
