# HashCraft – Cryptographic Hash Generator

**HashCraft** is a lightweight Python-based CLI tool designed to generate cryptographic hash values using multiple industry-standard hashing algorithms. It helps users understand password hashing, data integrity, and core cryptography concepts.

## 📅 Project Duration
**Sep 23, 2023 – Sep 28, 2023**

## ✨ Features

- Generate hashes using common algorithms:
  - MD5
  - SHA-1
  - SHA-256
  - SHA-512
- Interactive and easy-to-use **command-line interface**
- Hash user-provided strings or passwords
- Fast execution with minimal dependencies
- Beginner-friendly and educational

## 🚀 Usage

```bash
python hashcraft.py
