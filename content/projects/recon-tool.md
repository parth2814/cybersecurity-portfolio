# Automated Recon Tool

A comprehensive reconnaissance automation framework designed for penetration testers and security researchers.

## Features

- Automated subdomain enumeration
- Port scanning and service detection
- Vulnerability scanning integration
- Report generation
- Multi-threaded operations

## Installation

\`\`\`bash
pip install recon-tool
\`\`\`

## Usage

\`\`\`bash
recon-tool scan example.com
\`\`\`
